# 社区导航图标说明

## 需要准备的图标文件

请在 `static/images/tabbar/` 目录下准备以下两个图标文件：

### 1. community.png（未选中状态）
- 尺寸：81x81 像素（建议）
- 颜色：#6b7280（灰色）
- 图标：聊天气泡或社区图标
- 建议使用 iconfont 或 iconpark 等图标库下载

### 2. community_.png（选中状态）
- 尺寸：81x81 像素（建议）
- 颜色：#10b981（绿色）
- 图标：与 community.png 相同，但颜色为选中色
- 建议使用 iconfont 或 iconpark 等图标库下载

## 推荐图标样式

可以使用以下图标关键词搜索：
- chatbubble（聊天气泡）
- community（社区）
- people（人群）
- forum（论坛）
- message-circle（消息圈）

## 在线图标资源

1. **iconfont（阿里巴巴矢量图标库）**
   - 网址：https://www.iconfont.cn/
   - 搜索关键词：社区、论坛、聊天

2. **IconPark（字节跳动图标库）**
   - 网址：https://iconpark.oceanengine.com/
   - 搜索关键词：community、chat

3. **Flaticon**
   - 网址：https://www.flaticon.com/
   - 搜索关键词：community icon

## 临时方案

如果暂时没有图标，可以：
1. 复制 `work.png` 和 `work_.png`
2. 重命名为 `community.png` 和 `community_.png`
3. 后续再替换为合适的社区图标

## 图标设计建议

- 保持与其他导航图标风格一致
- 线条粗细统一
- 图标简洁明了
- 确保在小尺寸下清晰可辨
