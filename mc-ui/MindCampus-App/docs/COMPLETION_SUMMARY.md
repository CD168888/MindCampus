## 💡 使用技巧

### 修改主色调
只需修改一个变量：
```scss
// static/scss/theme.scss
$primary-color: #10b981;  // 当前翡翠绿
// $primary-color: #22c55e;  // 更鲜艳的绿色
// $primary-color: #14b8a6;  // 青绿色
```

### 调整心理评分
```javascript
// pages/index.vue
data() {
  return {
    mentalScore: 75,      // 修改此值（0-100）
    mentalLevel: '良好',  // 对应等级
  }
}
```

### 添加新模块
参照现有模块结构，保持统一风格：
```vue
<view class="my-section">
  <view class="section-header">
    <view class="section-title">🎯 新模块</view>
    <view class="section-more">更多 ›</view>
  </view>
  <!-- 内容 -->
</view>

<style lang="scss" scoped>
@import '@/static/scss/theme.scss';

.my-section {
  padding: 0 $spacing-lg $spacing-md;
}
</style>
```

## ⚠️ 注意事项

### 1. 颜色使用
- ✅ 主色用于关键操作
- ✅ 辅助色用于强调
- ✅ 中性色用于大面积背景
- ❌ 避免过度使用彩色

### 2. 间距统一
使用变量保持一致：
```scss
padding: $spacing-md;    // ✅ 统一
padding: 20rpx;          // ❌ 不推荐
```

### 3. 圆角统一
```scss
border-radius: $radius-md;    // ✅ 统一
border-radius: 15rpx;         // ❌ 不推荐
```

### 4. 阴影层次
```scss
box-shadow: $shadow-sm;     // 轻微
box-shadow: $shadow-base;   // 基础
box-shadow: $shadow-lg;     // 较强
```
