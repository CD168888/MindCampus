# 心理推荐管理模块创建说明

## 📝 功能概述

创建了三个心理推荐管理模块：
1. **心理音乐推荐** (RecommendMusic)
2. **心理文章推荐** (RecommendArticle)
3. **心理课程推荐** (RecommendCourse)

每个模块都包含完整的 CRUD 功能，支持查询、新增、修改、删除和导出。

## 📊 数据库表结构

### 1. recommend_music（心理音乐推荐表）

- `music_id` - 音乐ID（主键）
- `title` - 音乐标题
- `mp3_url` - 音频文件链接
- `cover_url` - 封面图链接
- `artist` - 演唱者/作者
- `genre` - 音乐风格
- `duration` - 时长（秒）
- `description` - 音乐简介
- `status` - 状态（0正常 1下架）

### 2. recommend_article（心理文章推荐表）

- `article_id` - 文章ID（主键）
- `title` - 文章标题
- `content` - 文章内容（Markdown格式）
- `summary` - 文章摘要
- `author` - 作者
- `read_count` - 阅读量
- `category` - 文章分类
- `status` - 状态（0正常 1下架）

### 3. recommend_course（心理课程推荐表）

- `course_id` - 课程ID（主键）
- `title` - 课程标题
- `mp4_url` - 视频文件链接
- `cover_url` - 封面图链接
- `lecturer` - 讲师
- `duration` - 视频时长（秒）
- `chapters` - 章节数
- `level` - 课程难度（初级/中级/高级）
- `description` - 课程简介
- `status` - 状态（0正常 1下架）

## 🔍 功能特性

### 1. 心理音乐推荐

- ✅ 支持上传 MP3 音频文件
- ✅ 支持上传封面图片
- ✅ 显示时长（格式：分:秒）
- ✅ 支持按标题、演唱者、风格、状态搜索
- ✅ 使用 `music_status` 字典显示状态

### 2. 心理文章推荐

- ✅ 支持 Markdown 格式内容编辑
- ✅ 显示阅读量统计
- ✅ 支持按标题、作者、分类、状态搜索
- ✅ 使用 `article_status` 字典显示状态

### 3. 心理课程推荐

- ✅ 支持上传 MP4 视频文件
- ✅ 支持上传封面图片
- ✅ 显示时长（格式：时:分:秒）
- ✅ 显示章节数
- ✅ 支持按标题、讲师、难度、状态搜索
- ✅ 使用 `course_status` 和 `course_level` 字典

## 📋 字典配置

需要在系统中配置以下字典：

### 1. music_status（音乐状态）
- 字典类型：`music_status`
- 字典值：
  - `0` = 正常
  - `1` = 下架

### 2. article_status（文章状态）
- 字典类型：`article_status`
- 字典值：
  - `0` = 正常
  - `1` = 下架

### 3. course_status（课程状态）
- 字典类型：`course_status`
- 字典值：
  - `0` = 正常
  - `1` = 下架

### 4. course_level（课程难度）
- 字典类型：`course_level`
- 字典值：
  - `初级` = 初级
  - `中级` = 中级
  - `高级` = 高级

## ⚠️ 注意事项

1. **文件上传**
   - MP3 文件大小限制：100MB
   - MP4 文件大小限制：500MB
   - 图片文件使用 ImageUpload 组件，默认限制由组件控制

2. **字典配置**
   - 必须配置所有相关字典
   - 字典值必须与代码中的值匹配

3. **权限配置**
   - 执行菜单 SQL 后，需要为角色分配相应权限
   - 前端按钮使用 `v-hasPermi` 指令控制显示

4. **时长显示**
   - 音乐时长：分:秒 格式
   - 课程时长：时:分:秒 格式（如果超过1小时）
