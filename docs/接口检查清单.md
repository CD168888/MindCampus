## ✅ 接口确认

### 1. Controller 文件已创建

**文件路径：** `mc-project/src/main/java/com/mc/community/controller/AppCommunityPostController.java`

**关键代码：**
```java
@RestController
@RequestMapping("/app/community/post")  // ← 基础路径
public class AppCommunityPostController extends BaseController {
    
    @GetMapping("/list")  // ← 完整路径：/app/community/post/list
    public TableDataInfo list(CommunityPost communityPost) {
        communityPost.setStatus("0");
        startPage();
        return getDataTable(communityPostService.selectCommunityPostList(communityPost));
    }
}
```

**完整接口路径：** `GET /app/community/post/list`

## 🚨 常见问题

### 问题 1：404 Not Found

**可能原因：**
- [ ] 服务未重启
- [ ] Controller 包路径不正确
- [ ] URL 拼写错误

**解决方法：**
1. 确认 Controller 在 `com.mc.community.controller` 包下
2. 重启服务
3. 检查 URL：`/app/community/post/list`（注意是 `/app/` 不是 `/community/`）

### 问题 2：No static resource

**可能原因：**
- [ ] Controller 未被 Spring 扫描到
- [ ] 缺少 `@RestController` 注解
- [ ] 缺少 `@RequestMapping` 注解

**解决方法：**
1. 确认类上有 `@RestController` 和 `@RequestMapping("/app/community/post")`
2. 确认方法上有 `@GetMapping("/list")`
3. 重启服务

### 问题 3：500 Internal Server Error

**可能原因：**
- [ ] 数据库表结构未更新
- [ ] SQL 执行错误
- [ ] 缺少必要字段

**解决方法：**
1. 执行 `sql/update_community_post_table.sql`
2. 检查数据库连接
3. 查看后端错误日志

### 问题 4：401 Unauthorized

**可能原因：**
- [ ] 未登录
- [ ] Token 过期
- [ ] Token 格式错误

**解决方法：**
1. 重新登录获取 token
2. 确认 Header 格式：`Authorization: Bearer YOUR_TOKEN`
3. 检查 token 是否有效
