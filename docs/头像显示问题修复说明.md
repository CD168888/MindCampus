# 头像显示问题修复说明

## 📋 问题描述

社区帖子和评论中，用户头像没有正常显示，即使用户已经上传了头像。

---

## 🔍 问题原因

1. **用户未上传头像**: 数据库中 `sys_user.avatar` 字段为 `NULL` 或空字符串
2. **前端判断不完善**: 空字符串 `""` 虽然为假值，但没有做 `trim()` 处理
3. **缺少默认头像**: 没有为未上传头像的用户提供默认头像

---

## ✅ 修复方案

### 1. 后端修改 - 提供默认头像

#### AppCommunityPostController.java

```java
// 设置用户头像，如果为空则使用默认头像
String avatar = sysUser.getAvatar();
if (avatar == null || avatar.trim().isEmpty()) {
    avatar = "/profile/avatar/default.png"; // 默认头像
}
communityPost.setUserAvatar(avatar);
```

#### AppCommunityCommentController.java

同样的修改应用于评论控制器。

---

### 2. 前端修改 - 完善判断逻辑

#### pages/community/index.vue

**修改前**:
```vue
<image v-if="item.userAvatar" 
       class="user-avatar" 
       :src="getImageUrl(item.userAvatar)" 
       mode="aspectFill">
</image>
```

**修改后**:
```vue
<image v-if="item.userAvatar && item.userAvatar.trim()" 
       class="user-avatar" 
       :src="getImageUrl(item.userAvatar)" 
       mode="aspectFill"
       @error="handleImageError">
</image>
```

**新增图片错误处理**:
```javascript
methods: {
  // 图片加载失败处理
  handleImageError(e) {
    console.error('头像加载失败:', e)
  }
}
```

### 方法2：创建新的默认头像

1. **放置位置**: `mc-admin/src/main/resources/static/profile/avatar/default.png`

2. **头像要求**:
   - 尺寸: 200x200 px（推荐）
   - 格式: PNG（支持透明背景）
   - 内容: 简洁的用户图标或字母

3. **示例代码生成简单头像**:

如果没有默认头像文件，可以使用以下方案：

**方案A**: 使用emoji或unicode字符作为默认显示
```vue
<!-- 前端修改 -->
<view v-else class="user-avatar-placeholder">
  <text style="font-size: 32rpx;">👤</text>
</view>
```

**方案B**: 使用用户昵称首字母
```vue
<view v-else class="user-avatar-placeholder">
  <text class="avatar-text">{{ getUserInitial(item.userName) }}</text>
</view>
```

```javascript
methods: {
  getUserInitial(name) {
    if (!name || name === '匿名用户') return '?'
    return name.charAt(0).toUpperCase()
  }
}
```

## 💡 优化建议

### 1. 头像加载优化

使用CDN或图片压缩服务加速头像加载：
```javascript
getImageUrl(url) {
  if (!url) return ''
  if (url.startsWith('http')) return url
  
  const baseUrl = config.baseUrl || 'http://localhost:8080'
  let fullUrl = url.startsWith('/') ? baseUrl + url : baseUrl + '/' + url
  
  // 添加图片尺寸参数（如果服务器支持）
  // fullUrl += '?w=100&h=100'
  
  return fullUrl
}
```

### 2. 头像缓存

在前端添加头像缓存机制：
```javascript
data() {
  return {
    avatarCache: {} // 头像缓存
  }
},
methods: {
  getImageUrl(url) {
    if (this.avatarCache[url]) {
      return this.avatarCache[url]
    }
    
    // 正常处理
    const fullUrl = this.processImageUrl(url)
    this.avatarCache[url] = fullUrl
    return fullUrl
  }
}
```

### 3. 懒加载

对于长列表，使用图片懒加载：
```vue
<image v-if="item.userAvatar && item.userAvatar.trim()" 
       class="user-avatar" 
       :src="getImageUrl(item.userAvatar)" 
       mode="aspectFill"
       lazy-load
       @error="handleImageError">
</image>
```
