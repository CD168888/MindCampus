## ❌ 常见错误及解决

### 错误 1：404 Not Found

**原因：** 服务未重启，接口未加载

**解决：**
```bash
# 停止服务
# 重新启动 McAdminApplication
```

### 错误 2：500 Internal Server Error

**原因：** 数据库表结构未更新

**解决：**
```sql
-- 执行数据库更新脚本
source sql/update_community_post_table.sql;
```

### 错误 3：No static resource app/community/post/list

**原因：** Controller 未被 Spring 扫描到

**检查：**
1. 文件路径是否正确：`com.mc.community.controller.AppCommunityPostController`
2. 类上是否有 `@RestController` 注解
3. 类上是否有 `@RequestMapping("/app/community/post")` 注解
4. 方法上是否有 `@GetMapping("/list")` 注解

**解决：** 重启服务

### 错误 4：401 Unauthorized

**原因：** Token 无效或未登录

**解决：**
1. 重新登录获取新 token
2. 检查 Header 格式：`Authorization: Bearer {token}`
