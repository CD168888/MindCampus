# 社区图片显示修复说明

## 问题描述

1. **图片不显示**：后端返回的 `images` 字段是逗号分隔的字符串，而前端代码期望的是数组
2. **图片布局单一**：缺少根据图片数量（1-9张）显示不同样式的布局

## 问题原因

### 数据格式问题

**后端返回的数据：**
```json
{
  "images": "http://localhost:8080/profile/upload/2025/11/10/xxx.png,http://localhost:8080/profile/upload/2025/11/10/yyy.jpg"
}
```

**前端期望的数据：**
```javascript
{
  images: [
    "http://localhost:8080/profile/upload/2025/11/10/xxx.png",
    "http://localhost:8080/profile/upload/2025/11/10/yyy.jpg"
  ]
}
```

## 解决方案

### 1. 首页列表（index.vue）

#### 数据处理
在 `loadPostList` 方法中添加了图片字符串转数组的处理：

```javascript
// 处理图片字段：将逗号分隔的字符串转换为数组
newList = newList.map(post => {
  if (post.images && typeof post.images === 'string') {
    post.images = post.images.split(',').filter(img => img.trim())
  } else if (!post.images) {
    post.images = []
  }
  return post
})
```

#### 模板优化
实现了根据图片数量显示不同布局：

- **1张图片**：大图展示（最大高度500rpx）
- **2张图片**：横向2列布局（每张220rpx高）
- **3张图片**：横向3列布局（每张180rpx高）
- **4张图片**：2×2网格布局（每张220rpx高）
- **5-9张图片**：3列网格布局（每张180rpx高）

#### 样式实现
```scss
/* 1张图片 - 大图展示 */
.images-layout-1 {
  .post-image {
    width: 100%;
    max-height: 500rpx;
    border-radius: $radius-base;
    background: $bg-gray-100;
  }
}

/* 2张图片 - 横向2列 */
.images-layout-2 {
  display: grid;
  grid-template-columns: repeat(2, 1fr);
  gap: $spacing-xs;
  
  .post-image {
    width: 100%;
    height: 220rpx;
    border-radius: $radius-sm;
    background: $bg-gray-100;
  }
}

// ... 其他布局样式
```

### 2. 详情页（detail.vue）

#### 数据处理
在 `loadPostDetail` 方法中添加了同样的处理逻辑：

```javascript
// 处理图片数组：将逗号分隔的字符串转换为数组
if (this.postDetail.images && typeof this.postDetail.images === 'string') {
  this.postDetail.images = this.postDetail.images.split(',').filter(img => img.trim())
} else if (!this.postDetail.images) {
  this.postDetail.images = []
}
```

#### 模板和样式
与首页列表保持一致的布局方案。

## 功能特性

1. ✅ **自动处理数据格式**：无论后端返回字符串还是数组，都能正确处理
2. ✅ **空值处理**：过滤掉空字符串和空格
3. ✅ **响应式布局**：使用 Grid 布局，自适应不同屏幕
4. ✅ **图片预览**：点击图片可预览，支持左右滑动
5. ✅ **最多9张**：自动限制最多显示9张图片
6. ✅ **统一样式**：首页和详情页保持一致的视觉效果
7. ✅ **加载占位**：灰色背景作为图片加载时的占位

## 注意事项

1. **图片URL处理**

## 技术要点

- **数据转换**：使用 `split(',').filter(img => img.trim())` 处理字符串
- **条件渲染**：使用 `v-if` 和 `v-else-if` 实现不同布局
- **Grid布局**：使用 CSS Grid 实现响应式网格
- **图片模式**：使用 `aspectFill` 保持比例裁剪
- **事件冒泡**：使用 `@tap.stop` 阻止事件冒泡
