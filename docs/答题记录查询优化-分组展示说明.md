# 答题记录查询优化 - 分组展示说明

## 📝 优化需求

根据 `result_id` 和 `questionnaire_id` 为一级分组，然后展示该组下的所有 `question_id`（题目答题记录）。同时通过关联查询获取学生姓名等信息。

## ✅ 优化内容

### 1. 数据库查询优化

**文件**：`mc-project/src/main/resources/mapper/questionnaire/QuestionnaireAnswerMapper.xml`

#### 1.1 关联查询优化

添加了完整的关联查询链路：

```sql
SELECT 
    qa.*,
    q.title as questionnaire_title,      -- 问卷标题
    er.student_id,                       -- 学生ID（通过 result_id 关联）
    s.name as student_name,              -- 学生姓名（通过 student_id 关联）
    qu.order_num as question_order_num   -- 题目顺序
FROM questionnaire_answer qa
LEFT JOIN evaluation_result er ON qa.result_id = er.result_id
LEFT JOIN student_info s ON er.student_id = s.student_id
LEFT JOIN questionnaire q ON qa.questionnaire_id = q.questionnaire_id
LEFT JOIN question qu ON qa.question_id = qu.question_id
```

**关联关系链**：
```
questionnaire_answer.result_id 
  → evaluation_result.result_id 
    → evaluation_result.student_id 
      → student_info.student_id 
        → student_info.name (学生姓名)

questionnaire_answer.questionnaire_id 
  → questionnaire.questionnaire_id 
    → questionnaire.title (问卷标题)

questionnaire_answer.question_id 
  → question.question_id 
    → question.order_num (题目顺序)
```

#### 1.2 排序优化

按分组和题目顺序排序：

```sql
ORDER BY 
    qa.result_id,           -- 一级：测评结果ID
    qa.questionnaire_id,     -- 一级：问卷ID
    qu.order_num,            -- 二级：题目顺序
    qa.question_id            -- 二级：题目ID
```

这样确保：
- 相同 `result_id + questionnaire_id` 的记录聚集在一起
- 同一组内的题目按 `order_num` 顺序排列

#### 2.2 合并单元格实现

使用 Element Plus 的 `span-method` 实现单元格合并：

```javascript
// 计算合并单元格
function handleSpanArr() {
  spanArr.value = []
  pos = 0
  
  for (let i = 0; i < questionnaireAnswerList.value.length; i++) {
    if (i === 0) {
      spanArr.value.push(1)
      pos = 0
    } else {
      const current = questionnaireAnswerList.value[i]
      const prev = questionnaireAnswerList.value[i - 1]
      
      // 判断是否属于同一组（result_id + questionnaire_id 相同）
      if (current.resultId === prev.resultId && 
          current.questionnaireId === prev.questionnaireId) {
        // 同一组，合并单元格
        spanArr.value[pos] += 1
        spanArr.value.push(0)
      } else {
        // 新组
        spanArr.value.push(1)
        pos = i
      }
    }
  }
}

// 表格合并单元格方法
function objectSpanMethod({ row, column, rowIndex, columnIndex }) {
  // 合并的列：测评结果ID(1)、学生姓名(2)、问卷标题(3)
  if (columnIndex === 1 || columnIndex === 2 || columnIndex === 3) {
    const _row = spanArr.value[rowIndex]
    const _col = _row > 0 ? 1 : 0
    return {
      rowspan: _row,
      colspan: _col
    }
  }
}
```

### 分组逻辑

- **一级分组**：`result_id + questionnaire_id`
  - 同一测评结果、同一问卷的所有答题记录为一组
  - 组内字段（resultId、studentName、questionnaireTitle）合并显示

- **二级展示**：`question_id`
  - 组内按题目顺序（order_num）排列
  - 每个题目一条记录，显示详细的答题信息

## 🔍 数据流程

```
1. 查询 questionnaire_answer 表
   ↓
2. 通过 result_id 关联 evaluation_result
   ↓ 获取 student_id
3. 通过 student_id 关联 student_info
   ↓ 获取学生姓名
4. 通过 questionnaire_id 关联 questionnaire
   ↓ 获取问卷标题
5. 通过 question_id 关联 question
   ↓ 获取题目顺序
6. 按 result_id, questionnaire_id, order_num 排序
   ↓
7. 前端按 result_id + questionnaire_id 分组
   ↓
8. 合并单元格展示
```
