# 社区帖子管理功能说明

## 功能概述

社区帖子管理功能提供了完整的后台管理界面，用于管理用户发布的社区帖子和评论。管理员可以查看、编辑、删除帖子，并查看帖子的所有评论（树状结构展示）。

## 功能特点

### 1. 帖子管理
- **列表展示**：显示所有帖子信息，包括学生姓名、内容、是否匿名、状态、评论数等
- **搜索筛选**：支持按学生ID、内容、是否匿名、状态等条件筛选
- **新增帖子**：管理员可以代用户新增帖子
- **编辑帖子**：修改帖子内容和状态
- **删除帖子**：删除帖子及其关联的所有评论（级联删除）
- **导出功能**：支持将帖子数据导出为Excel文件

### 2. 评论查看
- **详情对话框**：点击"详情"按钮查看帖子完整内容和所有评论
- **树状展示**：评论以树形结构展示（当前支持平铺列表，未来可扩展为多级回复）
- **评论信息**：显示评论者姓名（或匿名）、内容、发布时间、状态等

### 3. 数据字典
- **post_is_anonymous**：帖子是否匿名（0=否，1=是）
- **post_status**：帖子状态（0=正常，1=屏蔽）
- **comment_is_anonymous**：评论是否匿名（0=否，1=是）
- **comment_status**：评论状态（0=正常，1=屏蔽）

## 部署步骤

### 1. 数据库配置

已有表结构（无需重复创建）：
```sql
-- 社区帖子表
CREATE TABLE community_post (
  post_id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '帖子ID',
  student_id BIGINT COMMENT '学生ID',
  content TEXT COMMENT '帖子内容',
  is_anonymous CHAR(1) DEFAULT '1' COMMENT '是否匿名（0否 1是）',
  status CHAR(1) DEFAULT '0' COMMENT '状态（0正常 1屏蔽）',
  create_by VARCHAR(64) DEFAULT '' COMMENT '创建者',
  create_time DATETIME COMMENT '创建时间',
  update_by VARCHAR(64) DEFAULT '' COMMENT '更新者',
  update_time DATETIME COMMENT '更新时间',
  remark VARCHAR(200) DEFAULT NULL COMMENT '备注'
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='社区帖子表';

-- 社区评论表
CREATE TABLE community_comment (
  comment_id BIGINT PRIMARY KEY AUTO_INCREMENT COMMENT '评论ID',
  post_id BIGINT COMMENT '帖子ID',
  student_id BIGINT COMMENT '评论人ID',
  content VARCHAR(500) COMMENT '评论内容',
  is_anonymous CHAR(1) DEFAULT '1' COMMENT '是否匿名（0否 1是）',
  status CHAR(1) DEFAULT '0' COMMENT '状态（0正常 1屏蔽）',
  create_by VARCHAR(64) DEFAULT '' COMMENT '创建者',
  create_time DATETIME COMMENT '创建时间',
  update_by VARCHAR(64) DEFAULT '' COMMENT '更新者',
  update_time DATETIME COMMENT '更新时间',
  remark VARCHAR(200) DEFAULT NULL COMMENT '备注'
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='社区评论表';
```

执行菜单和字典配置SQL：

## 使用说明

### 1. 访问页面

登录后台管理系统后，在菜单中找到"帖子管理"（需要先执行SQL配置菜单权限）。

### 2. 查询帖子

- 输入搜索条件（学生ID、内容关键词、是否匿名、状态）
- 点击"搜索"按钮
- 点击"重置"清空搜索条件

### 3. 查看详情

- 点击列表中的"详情"按钮
- 对话框显示帖子完整信息
- 下方展示该帖子的所有评论（树状结构）
- 评论显示发布者、内容、时间、状态等信息

### 4. 编辑帖子

- 选中一条记录，点击"修改"按钮
- 修改表单中的字段
- 点击"确定"保存

### 5. 删除帖子

- 选中一条或多条记录
- 点击"删除"按钮
- 确认删除操作
- **注意**：删除帖子会同时删除该帖子的所有评论

### 6. 导出数据

- 设置筛选条件（可选）
- 点击"导出"按钮
- 下载Excel文件

### 3. 关键功能

#### 级联删除
删除帖子时自动删除关联的评论：
```java
@Override
@Transactional
public int deleteCommunityPostByPostIds(Long[] postIds) {
    // 先删除关联的评论
    communityCommentMapper.deleteCommunityCommentByPostIds(postIds);
    // 再删除帖子
    return communityPostMapper.deleteCommunityPostByPostIds(postIds);
}
```

#### 评论树状展示
前端使用 `el-tree` 组件展示评论：
```vue
<el-tree
  :data="commentTreeData"
  node-key="commentId"
  default-expand-all
  :expand-on-click-node="false"
>
  <template #default="{ node, data }">
    <!-- 评论内容展示 -->
  </template>
</el-tree>
```

#### 学生信息关联
通过左连接查询关联学生姓名：
```xml
<sql id="selectCommunityPostVo">
    select p.*, s.name as student_name,
           (select count(*) from community_comment c where c.post_id = p.post_id) as comment_count
    from community_post p
    left join student_info s on p.student_id = s.student_id
</sql>
```

## 扩展功能建议

### 1. 评论回复功能
可以在 `community_comment` 表中添加 `parent_id` 字段，实现多级回复：
```sql
ALTER TABLE community_comment ADD COLUMN parent_id BIGINT COMMENT '父评论ID';
```

然后在Service层实现递归构建树形结构。

### 2. 敏感词过滤
集成敏感词过滤库，自动检测和屏蔽违规内容。

### 3. 帖子点赞/收藏
添加点赞和收藏功能，增强用户互动。

### 4. 图片上传
支持帖子和评论上传图片。

## 常见问题

### 1. 菜单不显示
- 检查是否执行了 `community_dict.sql` 文件
- 检查用户角色是否有对应菜单权限

### 2. 数据不显示
- 检查数据库表是否存在
- 检查后端日志是否有错误
- 检查 MyBatis Mapper XML 路径是否正确

### 3. 删除失败
- 检查数据库外键约束
- 查看后端事务是否正常
