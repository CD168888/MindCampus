# MindCampus 高校心理健康智能干预平台

## 项目概述

MindCampus 是一个基于 Spring Boot 3.5.4 + MyBatis Plus + Vue 3 + Uni-app 的高校心理健康智能干预平台，集成阿里云 DashScope（通义千问）AI 能力，为高校学生提供心理健康测评、AI 心理辅导、危机干预等服务。

---

## 一、系统架构

### 1.1 技术栈

| 层级 | 技术选型 |
|------|----------|
| 后端框架 | Spring Boot 3.5.4 |
| ORM | MyBatis Plus |
| AI 能力 | Spring AI Alibaba + 阿里云 DashScope（通义千问） |
| 数据库 | MySQL 8.x |
| 缓存 | Redis |
| 前端 Web | Vue 3 + Vite + Pinia + Element Plus |
| 前端 App | Uni-app (Vue 3) |
| 安全认证 | JWT + Spring Security |

### 1.2 模块依赖关系

```
mc-admin (应用入口)
├── mc-framework (核心框架: 安全、AOP、配置)
├── mc-project (核心业务模块)
├── mc-system (系统管理)
├── mc-quartz (定时任务)
└── mc-generator (代码生成)

mc-project → mc-framework → mc-system → mc-common
```

## 五、关键功能说明

### 5.1 安全认证机制

- **JWT + Redis 认证**: Token 存储在 Redis 中，默认过期时间 30 分钟
- **RBAC 权限模型**: User → Role → Menu/Permission
- **数据权限**: @DataScope 注解实现部门级别数据隔离

### 5.2 AI 集成

- 使用 Spring AI Alibaba 集成阿里云 DashScope（通义千问）
- 支持流式 SSE 对话
- 对话历史存储在 Redis 中
- 提供工具函数（查学生信息、心理分析等）

### 5.3 危机干预流程

1. 学生完成心理测评 → 2. AI 分析评估结果 → 3. 根据风险等级配置触发干预通知 → 4. 通知辅导员/负责人 → 5. 处理记录跟踪

## 七、构建与运行

### 后端 (Java 17 + Maven)

```bash
# 清理项目
mvn clean

# 构建项目（跳过测试）
mvn clean package -Dmaven.test.skip=true

# 运行应用
java -jar mc-admin/target/mc-admin.jar
```

### 前端 - Web

```bash
cd mc-ui/MindCampus-Web
npm install
npm run dev     # 开发模式
npm run build:prod  # 生产构建
```

### 前端 - 移动端 App

使用 HBuilderX 打开 `mc-ui/MindCampus-App` 进行开发和打包。
